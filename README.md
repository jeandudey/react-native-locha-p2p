# react-native-locha-p2p

React Native API for the Locha Mesh P2P Networking

## Installation

```sh
npm install react-native-locha-p2p
```

## Usage

```js
import LochaP2P from "react-native-locha-p2p";

const result = await LochaP2P.hello("Locha");
```
